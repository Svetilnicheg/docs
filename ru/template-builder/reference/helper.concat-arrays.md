# helper.concat-arrays

Объединение нескольких массивов в один.

Например, у вас есть несколько массивов:

```json
([1, 2, 3], [4, 5, 6], [7, 8, 9])
```

Их элементы можно объединить в один массив, чтобы показать одновременно:

```json
[1, 2, 3, 4, 5, 6, 7, 8, 9]
```
